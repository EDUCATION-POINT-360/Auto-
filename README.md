# Education Point WhatsApp Auto-Reply Bot

This is a **server-based WhatsApp auto-reply bot** for Education Point and NUR.  
It replies like Muhammad Rizwan's personal assistant, sends group links, official platform links, and handles private/personal questions smartly.

## Features

- Auto-reply in Urdu / Roman Urdu
- Matric / Intermediate WhatsApp group links
- Official links sharing
- Personal/private question handling
- Friendly, professional, and mentor-style replies
